## Overview

Additional tools can be enabled for your agent to extend its capabilities. Each tool can be configured using the `additional_tools` parameter when running a task.

## Usage

To enable a tool, pass it in the `additional_tools` dictionary when creating a task:

<CodeGroup>
```python Python
from smooth import SmoothClient

client = SmoothClient(api_key="cmzr-YOUR_API_KEY")

result = client.run(
  task="Take a screenshot of the homepage",
  additional_tools={
    "screenshot": {"full_page": True},
    "hover": {}
  }
)
```
</CodeGroup>

## Available Tools

### Screenshot

Allows the agent to take screenshots of the browser viewport. The acquired screenshots can be accessed via `.downloads_url`.

**Parameters:**

<ResponseField name="full_page" type="boolean">
  Whether to take a full page screenshot rather than just the viewport. Default is `False`.
</ResponseField>

**Example:**

<CodeGroup>
```python Python
result = client.run(
  task="Take a full page screenshot",
  additional_tools={
    "screenshot": {"full_page": True}
  }
)
```
</CodeGroup>

### Hover

Allows the agent to hover over elements on the page. This tool has no configurable parameters.

**Example:**

<CodeGroup>
```python Python
result = client.run(
  task="Hover over the menu to reveal submenu items",
  additional_tools={
    "hover": {}
  }
)
```
</CodeGroup>
