## Launch a task using the Python SDK

Launch your first task in 4 lines of code. `client.run()` is a convenient shortcut for creating a temporary session and executing a single task.

<CodeGroup>

```python Python
# pip install smooth-py
from smooth import SmoothClient

client = SmoothClient(api_key="cmzr-YOUR_API_KEY")
task = client.run("Find the price of NVDA stock on Yahoo Finance")

task_result = task.result()  # Waits for the agent response

if task_result.status == "done":
  print(f"Agent response: {task_result.result}")
  print(f"Total cost: ${task_result.credits_used * 0.01}")
else:
  print(f"There was an error: {task_result.error}")
```

</CodeGroup>

<Note>
  For more complex workflows requiring multiple steps or manual interaction, use [Sessions](/methods/sessions).
</Note>


## Request Parameters

All parameters available when running a task via `client.run()`.

<ResponseField name="task" type="string" required>
  The task description for the agent to execute.
</ResponseField>

<ResponseField name="url" type="string">
  The starting URL for the task. If not provided, the agent will infer it.
</ResponseField>

<ResponseField name="response_model" type="dict | BaseModel">
  If provided, enforces a structured output schema. It can be a dictionary describing a JSON schema or a Pydantic Model.
</ResponseField>

<ResponseField name="profile_id" type="string">
  The ID of a [Browser Profile](/methods/profiles) to use. This allows the agent to use previously saved cookies and logins.
</ResponseField>

<ResponseField name="profile_read_only" type="boolean">
  If true, the profile will be loaded in read-only mode, and any changes (like new cookies) will not be saved. Default: `False`.
</ResponseField>

<ResponseField name="max_steps" type="int">
  Max number of steps the agent can take (max 64). Default: 32.
</ResponseField>

<ResponseField name="device" type="string">
  The type of device: `mobile` (default) or `desktop`.
</ResponseField>

<ResponseField name="enable_recording" type="boolean">
  Enable video recording of the task execution. Default: `True`.
</ResponseField>

<ResponseField name="stealth_mode" type="boolean">
  Run the browser in stealth mode to avoid detection. Default: `False`.
</ResponseField>

<ResponseField name="use_adblock" type="boolean">
  Enable adblock for the browser session. Default: `True`.
</ResponseField>

<ResponseField name="proxy_server" type="string">
  Proxy server address (e.g., `proxy.example.com`).
</ResponseField>

<ResponseField name="proxy_username" type="string">
  Proxy server username.
</ResponseField>

<ResponseField name="proxy_password" type="string">
  Proxy server password.
</ResponseField>

<ResponseField name="custom_tools" type="list">
  List of [Custom Tools](/methods/custom-tools) to register for this task.
</ResponseField>

<ResponseField name="additional_tools" type="dict">
  Dictionary of [Additional Tools](/methods/tools) to enable (e.g., `{"screenshot": {"full_page": True}}`).
</ResponseField>

<ResponseField name="extensions" type="list[str]">
  List of extension IDs to load into the browser for this task.
</ResponseField>

<ResponseField name="metadata" type="dict">
  A dictionary of variables that will be passed to the agent context.
</ResponseField>

## TaskHandle Methods

Methods available on the `TaskHandle` object returned by `client.run()`.

<ResponseField name="id()" type="method">
  Returns the ID of the task.
</ResponseField>

<ResponseField name="live_url(interactive=False, embed=False)" type="method">
  Returns a live URL where you can sessions the agent in action.
</ResponseField>

<ResponseField name="result(timeout=None)" type="method">
  Waits for task completion and returns a `TaskResponse`.
</ResponseField>

<ResponseField name="recording_url()" type="method">
  Returns the URL of the video recording after completion.
</ResponseField>

<ResponseField name="downloads_url()" type="method">
  Returns the URL of an archive containing any files downloaded.
</ResponseField>

## TaskResponse Attributes

The object returned by `task.result()`.

<ResponseField name="id" type="string">
  The ID of the task.
</ResponseField>

<ResponseField name="status" type="string">
  The status of the task. One of: `waiting`, `running`, `done`, `failed`, `cancelled`.
</ResponseField>

<ResponseField name="result" type="any">
  The final response from the agent.
</ResponseField>

<ResponseField name="credits_used" type="int">
  Amount of credits used (1 credit = $0.01).
</ResponseField>

<ResponseField name="error" type="string">
  Error message if the task failed.
</ResponseField>
