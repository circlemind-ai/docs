## Overview

Browser profiles allow you to persist cookies, local storage, and authentication state across multiple [sessions](/methods/sessions) and tasks.

## Create a browser profile

Create a new profile. You can provide a custom `profile_id` or let Smooth generate one for you.

<CodeGroup>

```python Python
from smooth import SmoothClient

smooth_client = SmoothClient(api_key="cmzr-YOUR_API_KEY")

# Create a profile with a specific ID
profile = smooth_client.create_profile(profile_id="my-authenticated-user")
print(f"Created profile: {profile.id}")
```

</CodeGroup>

## List all browser profiles

List all your existing browser profiles.

<CodeGroup>

```python Python
from smooth import SmoothClient

client = SmoothClient(api_key="cmzr-YOUR_API_KEY")
profiles = client.list_profiles()

# profiles is a list of ProfileResponse objects
for profile in profiles:
    print(profile.id)
```

</CodeGroup>

## Client Methods

The following methods are available on the `SmoothClient` and `SmoothAsyncClient` to manage profiles.

<ResponseField name="create_profile()" type="method">
  Creates a new browser profile.
</ResponseField>

<ResponseField name="list_profiles()" type="method">
  Returns a list of all existing browser profiles.
</ResponseField>

<ResponseField name="delete_profile()" type="method">
  Deletes an existing browser profile.
</ResponseField>

## Response Models

### ProfileResponse
<ResponseField name="id" type="string">
  The unique identifier of the browser profile.
</ResponseField>

<ResponseField name="created_at" type="number">
  The Unix timestamp when the profile was created.
</ResponseField>

## Delete a browser profile

Delete an existing browser profile and its associated data.

<CodeGroup>

```python Python
from smooth import SmoothClient

smooth_client = SmoothClient(api_key="cmzr-YOUR_API_KEY")
smooth_client.delete_profile(profile_id="my-profile-id")
```

</CodeGroup>
