## Overview

When running a task, you will receive a `live_url`, which can be used to view the agent actions live.

## Getting the URL

A `live_url` is available on the session object.

<CodeGroup>

```python Python
from smooth import SmoothClient

client = SmoothClient(api_key="cmzr-YOUR_API_KEY")

with client.session() as session:
    print(f"Live URL: {session.live_url()}")
    session.run_task("Go to Hacker News")
```

</CodeGroup>

<Note>
  If Python is not your language of choice, check out our [API Reference](/api-reference/introduction).
</Note>

## Customizations

You can customize the URL to make it interactive or full screen.

<CodeGroup>

```python Python
with client.session() as session:
    print(f"Live URL: {session.live_url(interactive=True, embed=True)}")
    session.run_task("Go to Hacker News")
```

</CodeGroup>
